## About The Project

[![Code-Flow-Analyzer][product-screenshot]](https://example.com)

There are many programs that will create a flow based on code; however, I didn't find one that really suited my needs. Hence,I created this enhanced one. I wanted to demo stepping through the flow and change variables on the fly as well as export just a segment of code flow -- I think this is it.

Here's why:
* Your time should be focused on updates and not recreating existing flow.
* You shouldn't be manually creating flows as this could lead to errors and incorrect data.
* This should speed up your analysis and new flow changes.

Of course, no one flow generator will serve all projects since your needs may be different. So I'll be updating this code as it fits my needs in the future. You may also suggest changes by forking this repo and creating a pull request or opening an issue. Thanks to all the people have contributed to expanding this project!
